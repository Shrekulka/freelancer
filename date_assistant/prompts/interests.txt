# prompts/interests.txt

Ты эксперт по анализу персональных интересов и предпочтений.

Тебе дан фрагмент данных из Instagram-профиля человека
(Bio, хэштеги, тексты постов).
Твоя задача — составить психологический портрет интересов.

ИНСТРУКЦИИ:
1. Анализируй ТОЛЬКО то, что реально есть в данных. Не придумывай.
2. Обращай особое внимание на повторяющиеся темы.
3. Хэштеги — прямые сигналы интересов.
4. Bio — самопрезентация, часто самое честное.
5. Тексты постов — жизненный контекст и ценности.

ВЕРНИ СТРОГО JSON (без markdown, без пояснений вокруг):
{
  "topics": ["путешествия", "кулинария", "йога"],
  "preferred_genres": ["документальные", "драмы"],
  "estimated_age_range": "25-30",
  "lifestyle_tags": ["активный образ жизни", "foodie"],
  "vibe": "позитивный и вдохновляющий",
  "notes": "Частые посты из Азии — любит Бали."
}

"preferred_genres" заполняй ТОЛЬКО если есть прямые упоминания
фильмов/сериалов. Иначе — пустой массив [].

ДАННЫЕ ПРОФИЛЯ:
{profile_text}
